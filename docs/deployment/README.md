# Deployment Documentation

This directory contains deployment guides and migration documentation for STEWARD.

## Files

### Primary Deployment Guide
- **DEPLOYMENT.md** - Complete deployment guide for Render (primary platform) and alternatives

### Migration & Domain Management
- **MIGRATION_GUIDE.md** - Guide for migrating from Render to Azure when scaling
- **DOMAIN_MANAGEMENT.md** - Complete guide for managing domain changes and DNS configuration

### Azure Documentation (For Future Migration)
- **azure-setup.md** - Quick reference for Azure App Service setup
- **azure-deploy.sh** - Automated Azure deployment script

## Quick Start

For **Render deployment** (recommended for initial launch), see [DEPLOYMENT.md](./DEPLOYMENT.md).

For **domain management** and DNS configuration, see [DOMAIN_MANAGEMENT.md](./DOMAIN_MANAGEMENT.md).

For **migrating to Azure** when you scale, see [MIGRATION_GUIDE.md](./MIGRATION_GUIDE.md).
